## 题目二

像素雨游戏：

请用HTML元素(非Canvas,WebGL)方式实现。 

实现细节,红色10X10方块从屏幕上部以随机的1-10像素的速度从屏幕上方垂直漂落至屏幕下方，速度越快，透明度越低。如果漂出屏幕则重置回屏幕上方，让他们LOOP起来。默认100红色方块，之后每次鼠标点击加入100个方块。请将帧率统计出来写在页面的右上角，得分就是方块数*帧率。
